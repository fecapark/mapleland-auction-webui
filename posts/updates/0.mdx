안녕하세요.

메이플랜드 옥션 운영팀입니다.

<br />

메이플랜드 옥션은 현재 1인 개발 및 운영의 체제로 진행되고 있습니다.  
따라서 많은 기능을 단시간내에 개발하여 제공하기 어려운 점 양해 부탁드립니다.

<br />

향후 개발 예정인 기능들에 대해 미리 공지드리겠습니다.

### 1. 장비 아이템 카테고리 시세 검색

장비 아이템들에 대한 시세 검색을 지원합니다.

주문서 사용 남은 횟수, N상/하옵 등을 선택하여 세부적으로 시세를 검색할 수 있도록 합니다.

<br />

단, 디스코드 거래 메시지의 경우 형식 자체가 소비/기타 아이템과는 달리 너무나도 다양하여,

우선적으로 메랜지지의 시세부터 지원할 계획이며, 이후에 디스코드 거래 메시지에 대한 시세 검색을 지원할 예정입니다.

### 2. 아이템 즐겨찾기 / 아이템 검색 기록

찾고자하는 아이템을 즐겨찾기에 추가하여 홈 화면에서 즉시 시세를 확인할 수 있도록 합니다.

또한, 검색하였던 아이템들에 대한 검색 기록을 확인할 수 있도록 합니다.

### 3. 시세 분포 확인 기능

아이템 시세 페이지에서 현재 시세의 분포를 확인할 수 있도록 합니다.

그래프의 형태로 제공될 예정이며, 이를 통해 시세의 밀집도를 확인할 수 있도록 합니다.

### 4. 트렌딩 섹션 추가

'1일 간' 혹은 '어제 대비' 변화량이 큰 아이템들을 트렌딩 섹션에 추가하여 확인할 수 있도록 합니다.

<br />

이 외에도, 여러가지 지표를 적용한 트렌딩 섹션을 추가할 계획입니다.

### 5. 시세 변동 차트 개선

현재 아이템 시세를 확인하는 차트의 형태는 단순한 점선 그래프입니다.

<br />

물론 점선 그래프가 나쁘다는 것은 아니지만, 데이터의 시간 단위를 변화시켜 보여 줄 경우 사용자로 하여금 혼란을 초래할 수 있습니다. (1시간 단위로 측정되는 데이터를 1일 단위로 바꿀 경우, 데이터 계산의 기준을 잡기 애매해짐)

<br />

많은 개선점을 생각중이며, 내부 계산 로직의 개선으로 충분히 개선이 가능하다면 차트의 형태는 유지하되, 내부 계산 로직을 개선할 계획입니다.

<br />

만약 로직의 개선으로도 원하는 만큼의 개선이 어렵다면, 차트의 형태를 변경할 계획입니다.

계획중인 차트의 형태는 캔들 차트의 형태입니다.

### 6. 시세 가중치 적용 기능

시세 검색 시, 특정 거래소 / 특정 아이템 (또는 아이템 종류) 의 시세에 가중치를 적용하여 시세를 확인할 수 있도록 합니다.

<br />

모든 사용자에게 보이는 것이 아닌, 사용자 개인에게만 보이도록 설정되며,{" "}

이를 통해 사용자 개인의 거래소별 선호도를 반영하여 시세를 확인할 수 있도록 합니다.

<br />

하지만 이 기능은 '스크린샷' 기능을 통해 시세를 조작하여 유포할 수 있을 가능성이 있는 기능이므로,

여러 가지 방법을 통해 이를 방지할 수 있도록 조치 할 예정이며, 막을 수 있는 가능성이 없다고 판단될 시 폐기될 기능입니다.

심사 숙고를 거쳐 업데이트를 진행하도록 하겠습니다.

<br />
<br />

추가되는 기능들에 대해서는 개발이 지연되거나 변경될 수 있다는 점 다시 한 번 양해 부탁드립니다.

기능과 관련된 건의사항이 있다면, 언제든지 문의해주시면 감사하겠습니다.
